## Основы

h - влево (самая левая кнопка)
j - вниз (похожа на стрелку вниз)
k - вверх
l - вправо (самая правая кнопка)

wq - write quit
wqa - write quit all

v - visual mode
V - visual mode with selected line

u - undo
Ctrl + r - redo

## Copy-paste
- выделить кусок текста в visual mode
- y (yank) - put text in register
- p (put, pase)
